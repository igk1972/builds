# builds

Custom builds of upstream software, made **as needed for my personal projects** —
compiled or patched the way I want them (musl/Alpine, extra plugins, specific
toolchains) when an upstream artifact doesn't exist or doesn't fit.

They're public, so **feel free to use them for your own projects** — but they're
maintained on a best-effort basis for my needs, with no stability or support
guarantees. Pin versions and test before relying on them.
